## ADDED Requirements

### Requirement: 카테고리 accordion에서 여러 카테고리를 동시에 펼칠 수 있다
카테고리별 지출 탭에서 각 카테고리의 accordion은 독립적으로 열리고 닫혀야 한다. 하나를 열었을 때 다른 카테고리가 닫히지 않아야 한다.

#### Scenario: 여러 카테고리 동시 열기
- **WHEN** 사용자가 이미 펼쳐진 카테고리가 있는 상태에서 다른 카테고리의 펼치기 버튼을 클릭하면
- **THEN** 기존 카테고리는 그대로 열린 상태를 유지하고 새 카테고리도 함께 펼쳐진다

#### Scenario: 개별 카테고리 닫기
- **WHEN** 사용자가 이미 펼쳐진 카테고리의 펼치기 버튼을 다시 클릭하면
- **THEN** 해당 카테고리만 닫히고 나머지 열린 카테고리는 유지된다

### Requirement: 사용자는 결제 건을 선택해 원하는 항목의 합산 금액을 확인할 수 있다
카테고리 accordion 내 결제 건 클릭 시 선택 상태로 전환되고, 여러 카테고리에 걸쳐 다중 선택이 가능해야 한다. 선택된 건들의 금액 합산이 실시간으로 표시된다.

#### Scenario: 단일 결제 건 선택
- **WHEN** 사용자가 펼쳐진 accordion 내 결제 건을 클릭하면
- **THEN** 해당 결제 건이 선택 상태(하이라이트)로 표시되고 선택 패널에 금액이 반영된다

#### Scenario: 여러 카테고리에 걸친 다중 선택
- **WHEN** 사용자가 두 개 이상의 카테고리를 동시에 열고 각 카테고리에서 결제 건을 선택하면
- **THEN** 선택된 모든 건이 유지되고 합산 금액이 합산 패널에 즉시 갱신된다

#### Scenario: 선택 해제
- **WHEN** 사용자가 이미 선택된 결제 건을 다시 클릭하면
- **THEN** 해당 건의 선택이 해제되고 합산 금액이 즉시 갱신된다

### Requirement: 선택 합산 결과가 레이아웃에 맞게 표시된다
선택 건수가 1 이상이면 데스크톱에서는 우측 고정 패널(280px)로, 모바일에서는 하단 sticky 바로 표시된다.

#### Scenario: 데스크톱 사이드 패널 표시
- **WHEN** 데스크톱(≥ 1024px) 환경에서 결제 건을 1건 이상 선택하면
- **THEN** 카테고리 목록 오른쪽에 고정된 사이드 패널에 선택 건수, 항목 리스트, 합산 금액이 표시된다

#### Scenario: 모바일 하단 바 표시
- **WHEN** 모바일(< 1024px) 환경에서 결제 건을 1건 이상 선택하면
- **THEN** 화면 하단에 sticky 바가 나타나 선택 건수와 합산 금액이 한 줄로 표시된다

#### Scenario: 선택 없을 때 패널 미표시
- **WHEN** 선택된 결제 건이 없으면
- **THEN** 데스크톱 사이드 패널과 모바일 sticky 바가 모두 숨겨진다 (모바일 바는 hidden, 데스크톱은 빈 상태 안내)

### Requirement: 선택된 결제 건들의 금액 합산이 실시간으로 계산된다
`selectedIds: Set<string>` 상태를 기반으로 전체 `transactions` 배열을 필터링해 `selectedTransactions`와 `selectedTotal`을 도출해야 한다. 이 값들이 SelectionPanel과 StickySelectionBar에 전달된다.

#### Scenario: 합산 금액 실시간 갱신
- **WHEN** 사용자가 결제 건을 선택하거나 해제하면
- **THEN** `selectedTotal`이 즉시 재계산되어 SelectionPanel과 StickySelectionBar에 반영된다

#### Scenario: 취소 거래 선택 시 합산
- **WHEN** 사용자가 취소 거래(isCancel: true)를 선택하면
- **THEN** 해당 금액(음수)도 합산에 포함된다 (명시적 선택이므로 제외하지 않음)

### Requirement: 초기화 버튼으로 모든 선택을 한 번에 해제할 수 있다
선택 패널(데스크톱) 또는 모바일 sticky 바에 초기화 버튼이 있어야 한다.

#### Scenario: 선택 초기화
- **WHEN** 사용자가 초기화 버튼을 클릭하면
- **THEN** 모든 카테고리에 걸쳐 선택된 결제 건이 일괄 해제되고 합산 패널이 초기 상태로 돌아간다
